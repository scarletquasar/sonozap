# Sonozap
Whatsapp Web clone monorepo (backend and frontend).

# Stack

- Frontend
    - TypeScript
    - React.js
    - Relay
    - Zod

- Backend
    - TypeScript
    - Express.js
    - GraphQL
    - Drizzle
    - Zod
